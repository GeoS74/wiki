# Указатели

### Ключевые моменты:

- **Указатель — это переменная, хранящая адрес.** Сама по себе она не создаёт память для данных.
- **Перед использованием указателя он должен указывать на валидную память:**
    - На существующий массив (`p = arr`)
    - На динамически выделенную память (`p = malloc(...)`)
    - На строковый литерал (`p = "hello"`) - но только для чтения
- **Нельзя копировать данные в неинициализированный указатель:**

```c
char *p;        // p содержит мусор
strcpy(p, arr); // запись по случайному адресу — катастрофа
```

- **Разница между указателем и массивом:**
    - Массив - это область памяти с данными
    - Указатель - это переменная, которая может хранить адрес такой области
- **Строковые литералы - хранятся в read-only памяти**, их нельзя изменять.
