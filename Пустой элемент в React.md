# Пустой элемент в React

Пустой элемент нужен для рендеринга нескольких html тегов одним компонентом в [[React]]. При этом, после рендеринга никакой html блок (обёртка) фактически не создаётся.

```js
function App() {
  return (
    <>
      <h1>hello world</h1>
      <p>lorem ipsum</p>
    </>
  );
}
```
