# Агрегация

Чтобы использовать агрегацию в $lookup:

```
...
{
	$lookup: {
        from: 'prices',
        localField: '_id',
        foreignField: 'nomenclatureId',
        as: 'price',

        pipeline: [{
          $match: {
            createdAt: {$lt: new Date(Date.now() - 1000 * 3600 * 24 * 5)}
          }
        }]
      },
},
...
```
