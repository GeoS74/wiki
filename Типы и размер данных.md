
# Типы и размер данных в [[Си]]

Именованные константы для всех размеров вместе с другими характеристиками машины и компилятора содержатся в заголовочных файлах <limits.h> и<float.h>.

Файл <limits.h> находится в директории `/usr/include/limits.h`

Файл <float.h> находится в другом месте, чтобы его найти, выполни:

```bash
find /usr -name "float.h" 2>/dev/null
```


В файле limits.h можно найти конкретные значения, а в файле float.h только макросы. Это сделано по следующим причинам.

**Для целых чисел (`limits.h`):**
- Размеры фиксированы архитектурой
- `int` всегда 32 бита на x86_64 Linux
- Можно задать константы напрямую

**Для плавающей точки (`float.h`):**
- Зависит от целевой платформы (x86, ARM, PowerPC)
- Зависит от настроек компилятора (`-mfpmath=sse`, `-march=`)
- Зависит от стандарта (IEEE 754-1985 vs IEEE 754-2008)

>Значения в файле limits.h можно поменять вручную. Скорее всего это сломает систему или компилятор или ещё что-то. Надо попробовать в [[Docker]] контейнере.
