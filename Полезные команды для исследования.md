# Полезные команды для исследования

Предполагается, что есть файл с кодом `types.c`, который [[Компиляция программ Си|компилируется]] командой:

```bash
gcc types.s -o types
```


Посмотреть, как компилятор представляет код:

```bash
gcc -S types.c # создаст types.s (ассемблер)
```

Посмотреть секции исполняемого файла:

```bash
size ./types
```

Шестнадцатиричный дамп данных:

```bash
hexdump -C ./types
```

Смещение переменных в структуре:

```bash
gcc -dM -E - < /dev/null | grep -i align
```
