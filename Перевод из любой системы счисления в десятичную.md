
# Перевод из любой системы счисления в десятичную

#### hex to integer

Обобщённая формула перевода hex в десятеричную систему счисления:
decimal = dₙ₋₁×16ⁿ⁻¹ + ... + d₂×16² d₁×16¹ + d₀×16⁰

Тут есть проблема: на каждом шаге итерации требуется возведение в степень, а это приводит к усложнению сложности алгоритма. Сложность равна O(n²).

Для упрощения сложности можно использовать алгоритм Хорнера.

#### Алгоритм Хорнера для перевода из любой системы счисления в десятичную

Обобщённый алгоритм:
число₁₀ = ((...(d₀×base + d₁)×base + d₂)×base + ... + dₙ₋₁)

где:
- `dᵢ` — значение i-й цифры (слева направо)
- `base` — основание системы счисления (для hex = 16)
- `n` — количество цифр

Псевдокод:
```
result = 0
для каждой цифры c слева направо:
    digit = значение_цифры(c)  # 0-9, 10-15 для hex
    result = result * base + digit
вернуть result
```

Пример для hex:

```
"0xAF" → ?
result = 0
'A' → result = 0×16 + 10 = 10
'F' → result = 10×16 + 15 = 175
```

#### Преимущества алгоритма Хорнера:

1. Одно умножение на цифру (O(n) вместо O(n²))
2. Нет вычисления степеней → нет переполнения промежуточных результатов
3. Работает слева направо как читается число
4. Универсален для любой системы счисления
