# Пользователи


### Список пользователей

```bash
cat /etc/passwd

# Более читаемо с форматированием
cat /etc/passwd | column -t -s:
```

### Добавить пользователя

```bash
sudo adduser <username>

# добавить пользователя без пароля
sudo adduser --disabled-password <username>
```
### Удалить пользователя

```bash
# Основная команда (удалит пользователя и домашнюю папку)
sudo userdel -r <username>

# Принудительное удаление, если пользователь в системе
sudo pkill -u <username> 2>/dev/null
sudo userdel -f -r <username>
```
