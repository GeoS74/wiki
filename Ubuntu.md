Поиск по командам [[Docker]]-a и их описаниям совпадения с "cont":
```
$ docker help | grep cont
```

Запуск программы в фоновом режиме:
```
$ programmName &
```
В терминал будет выведен id процесса, который можно затем завершить так:
 ```
 $ kill idProcess
 ```
